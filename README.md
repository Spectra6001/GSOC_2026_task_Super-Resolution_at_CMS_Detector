# Super Resolution at CMS Detector
## Project Overview
This repository contains my submission for the GSoC 2025 project **"Super Resolution at CMS Detector"** under **ML4SCI**. This project focuses on integrating deep learning super resolution techniques to conver low-resolution calurimeter data to high-resolution data, without violating physical constraints.
## Repository Structure
- **`/Dataset/`**: Contains the dataset link provided.
- **`/Specific Task/`**: Contains implementation for Super Resolution task using GANs with three different generator architectures and custom physics constrained loss.
- **`/Initiative/`**: Contains the implementation of a Super Resolution Diffusion model taken as a initiative for the project to understand diffusion model architectural structures and stability.
## Project Task
Super Resolution of low-resolution images impinging on a calorimeter to high-resolution images and Discuss the choices made in model selection and optimization.
